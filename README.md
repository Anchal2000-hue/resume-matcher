# Resume Job Matcher 🎯

An AI-powered web app that analyzes your resume against a job description and gives you a match score with actionable feedback.

## Features

- 📊 Match score from 0-100%
- ✅ Matching keywords between resume and job
- ❌ Missing keywords you should add
- 💡 3 actionable suggestions to improve your resume
- 📝 One-line summary of your match

## Tech Stack

- **Python** + **Flask** (backend)
- **Groq AI** + **LLaMA 3.3** (AI analysis)
- **HTML/CSS/JavaScript** (frontend)

## Run Locally

1. Clone the repo
```bash
   git clone https://github.com/Anchal2000-hue/resume-matcher.git
   cd resume-matcher
```

2. Install dependencies
```bash
   pip install -r requirements.txt
```

3. Create `.env` file
4. Run the app
```bash
   python app.py
```

5. Open http://127.0.0.1:5000

## How It Works

1. Paste your resume text
2. Paste the job description
3. Click **Analyze Match**
4. Get your AI-powered score and feedback instantly!
